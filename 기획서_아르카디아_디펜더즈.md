# 🏰 아르카디아 디펜더즈 (Arcadia Defenders)
## 팔라독 스타일 카드 디펜스 게임 - 기획서

### 세계관 컨셉
**아르카디아** - 마법과 모험의 판타지 대륙. 어둠의 군단이 침공하자, 영웅과 길들인 몬스터를 카드에 봉인하여 소환하는 "카드 소환사"가 대륙을 지킨다.

- 저작권 문제 없는 **오리지널 판타지 세계관**
- 이미지는 **이모지 + CSS 그래픽**으로 구현 (외부 에셋 불필요)
- 추후 도트 스프라이트로 업그레이드 가능

---

## 1. 게임 개요
- **장르**: 횡스크롤 카드 디펜스 (팔라독 스타일)
- **플랫폼**: 웹 브라우저 (React + Canvas)
- **모드**: 싱글 우선 → 추후 멀티/과금
- **비주얼**: 이모지 유닛 + CSS UI + Canvas 전투

---

## 2. 핵심 루프

```
메인화면 → 덱편성 → 스테이지선택 → 전투 → 보상 → 카드수집/강화 → 반복
```

## 4. 전투 시스템

### 4-1. 기본 구조
- 횡스크롤: 왼쪽(아군 기지) ↔ 오른쪽(적 기지)
- Canvas로 렌더링 (이모지 유닛이 이동/전투)
- 실시간 진행 (requestAnimationFrame)

### 4-2. 마나 시스템
- 시작 마나: 10
- 최대 마나: 100
- 회복: 초당 2
- 소환 코스트: 카드 cost × 3

### 4-3. 전투 UI (Canvas)
```
[아군기지HP] ←아군유닛들→ ←적유닛들→ [적기지HP]
[마나바] [소환카드슬롯 1~5] [자동전투] [배속]
```

### 4-4. 승리 조건
- 적 기지 HP 0 → 승리 (1~3성)
- 아군 기지 HP 0 → 패배

## 6. 카드 획득 (싱글 - 과금 없음)

### 6-1. 스테이지 드랍
- 클리어 시 골드 + 경험치
- 확률 카드 드랍 (스테이지별 테이블)

### 6-2. 골드 뽑기
- 1회: 200골드 / 10연: 1,800골드
- 천장: 30회 → ★★★ 이상 보장

### 6-3. 상점
- 골드로 특정 카드 직접 구매
- 로테이션 상점 (매 플레이마다 변경)

### 6-4. 강화
- 레벨업: 중복 카드 재료 → 스탯 증가
- 합성: 동일 카드 N장 → 상위 등급 진화

### 구현 방식
| 요소 | 구현 방법 |
|---|---|
| 유닛/몬스터 | 픽셀아트 스프라이트 (에셋팩) + 등급별 테두리 효과 |
| 배경 | 픽셀아트 타일맵 (챕터별 테마) |
| HP바 | CSS 또는 Canvas 사각형 |
| 카드 UI | 픽셀아트 프레임 + CSS (등급별 색상 테두리) |
| 이펙트 | Canvas 파티클 (폭발, 힐, 버프) |
| 가챠 연출 | CSS 애니메이션 (카드 뒤집기, 빛 효과) |
| UI 아이콘 | 픽셀아트 아이콘팩 (골드, 마나, 별 등) |

### 에셋 폴더 구조
```
public/assets/
├── units/          # 유닛 스프라이트 (idle, attack, walk)
├── monsters/       # 몬스터 스프라이트
├── backgrounds/    # 스테이지 배경
├── ui/             # UI 프레임, 아이콘, 버튼
├── effects/        # 이펙트 스프라이트
└── cards/          # 카드 프레임 (등급별)
```
